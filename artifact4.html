Artifact 4: The ePortfolio

Overview

This ePortfolio is a culmination of my learning and growth in COMP 2511. Built with semantic HTML, custom CSS, and responsive design principles, it showcases my technical and design expertise.

Key Features

  • Responsive Design: Optimized for all screen sizes using CSS Grid and Flexbox.
  • Interactive Elements: Buttons, hover effects, and navigation links enhance user experience.
  • Accessibility: Features alt textand keyboard-friendly navigation.
  • Custom Styling: A sleek and easy to follow light theme with vibrant accent colors.

Challenges and Solutions

Maintaining Consistency Across Pages

To ensure consistent styling, I implemented a global CSS file and tested on multiple browsers. Using reusable components like a shared header and footer streamlined development.

Responsive Layout Issues

Initial designs did not scale well for smaller screens. I fixed this by refining media queries and using a mobile-first approach for layouts.

Workflow and Learning Outcomes

This project refined my understanding of semantic HTML and responsive design. The iterative development process helped me troubleshoot issues and develop clean, maintainable code.

Beyond technical skills, crafting this ePortfolio gave me insights into effective communication and user-centric design. I'm proud to present this as a representation of my journey.
